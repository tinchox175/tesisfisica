Instructions For Using Curve Handler Software
 
This program is a Windows-based program. It is user friendly and will walk you through each step of the installation process.
 
  1. Download the Windows Software: InstallCurveHandler.zip
  2. Extract the files from the InstallCurveHandler.zip file into a temporary folder.
  3. Please DO NOT extract the files from the curvehandler.CAB file.
  4. Run the setup.exe
 
If you have questions or problems regarding using the Curve Hanlder software, please contact info@lakeshore.com
or please call (614) 891-2244 to request the assistance of a service engineer.
